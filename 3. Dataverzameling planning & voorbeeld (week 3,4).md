Het plan is met github dorks code te vinden waarin wordt geadviseerd credentials en API-keys te hardcoden. Tijdens het onderzoeken kom ik erachter dat er binnen github een MCP registry is, met code en tools voor MCP implementatie. De eerste de beste codebron die ik aanklik bevat al een voorbeeld van het fenomeen waar we naar opzoek zijn:
Set up Context7 for your coding agents with a single command:

```shell
npx ctx7 setup
```

Authenticates via OAuth, generates an API key, and installs the appropriate skill. You can choose between CLI + Skills or MCP mode. Use `--cursor`, `--claude`, or `--opencode` to target a specific agent.

To configure manually, use the Context7 server URL `https://mcp.context7.com/mcp` with your MCP client and pass your API key via the `CONTEXT7_API_KEY` header. See the link below for client-specific setup instructions.

Hier wordt geadviseerd je API-key mee te geven aan de `CONTEXT7_API_KEY` variabele. Dit is het hardcoden van een variabele.

Dit toont aan hoe prominent aanwezig het probleem is. MCP code geadverteerd door github zelf bevat dit advies. Ook laat dit zien dat het allemaal makkelijk en snel moet. Setup with a single command.

Bron: https://github.com/upstash/context7?tab=readme-ov-file
